# 🐍 贪吃蛇游戏 (Snake Game)

这是一个使用原生 HTML、CSS 和 JavaScript 开发的经典贪吃蛇游戏。界面现代，操作流畅，支持速度调节和最高分记录。

## ✨ 功能特性

- **经典玩法**：控制蛇吃食物，身体变长，避免撞墙或撞到自己。
- **游戏控制**：
  - ▶️ **开始**：开始游戏
  - ⏸️ **暂停**：暂停/继续游戏
  - 🔄 **重置**：重新开始游戏
- **速度调节**：通过滑块实时调整游戏速度（1-20 级）。
- **分数记录**：
  - 实时显示当前分数。
  - 自动保存最高分（使用本地存储）。
- **响应式设计**：美观的界面，适配不同屏幕尺寸。

## 🎮 游戏操作

- **移动**：
  - 使用 **方向键** (⬆️ ⬇️ ⬅️ ➡️)
  - 或 **WASD** 键 (W: 上, S: 下, A: 左, D: 右)
- **空格键**：暂停/继续游戏

## 🚀 如何运行

1. 克隆或下载本项目到本地。
2. 直接在浏览器中打开 `index.html` 文件即可开始游戏。

## 🛠️ 技术栈

- **HTML5**: 游戏结构和 Canvas 绘图容器。
- **CSS3**: 现代化的 UI 设计，使用 Flexbox 布局和渐变背景。
- **JavaScript**: 游戏核心逻辑，包括 Canvas 渲染、碰撞检测和状态管理。
